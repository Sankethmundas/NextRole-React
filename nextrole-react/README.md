# NextRole React

NextRole React is a modern career management platform built with React.js. It helps job seekers create professional resumes, track applications, generate cover letters, and improve their chances of landing their next role.

## Features

### Resume Builder

* Create ATS-friendly resumes
* Live resume preview
* Download resume as PDF
* Multiple resume templates

### Job Tracker

* Track job applications
* Monitor application status
* Organize company and role details
* Manage interview progress

### ATS Resume Checker

* Analyze resume content
* Check keyword matching
* Improve ATS compatibility
* Receive resume improvement suggestions

### Cover Letter Generator

* Generate professional cover letters
* Multiple templates
* Copy and download functionality
* Customize for different job roles

### Pricing Page

* Free, Pro, and Premium plans
* Responsive pricing cards
* Modern UI design

### Authentication Pages

* Login
* Register
